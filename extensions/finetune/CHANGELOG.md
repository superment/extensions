# FineTune Changelog

## [Initial Release] - 2026-02-21

- Output Devices command: list apps and output devices, set volume, mute, reset, route to device
- Reset All to 100% command: reset all apps to 100% volume and unmute
- Per-app volume presets and custom volume
- Per-app mute and volume up/down
- Route to Device with device icons (Monitor, AirPods, Headphones, Speaker)
- Per-device volume control for supported output devices
- Optional preferences: Default device UID, Device UID overrides (for devices like Studio Display)
- Requires FineTune app; empty state with install links when missing
